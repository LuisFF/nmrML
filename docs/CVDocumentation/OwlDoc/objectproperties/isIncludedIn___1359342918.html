Object Property: 'is included in'

http://purl.org/biotop/btl2.owl#isIncludedIn

Annotations (3)

  • definition "includedIn (inverse: includes) is the most general relation which relates one (spatio)temporally qualified entity with another (spatio)temporally qualified entity where it occurs, inheres, or is part of. Example: A heart pumping process is included in a heart. A day is included in a month John" ()
  • definition "isIncludedIn is a very broad relation which relates an entity with the place it occurs, inheres, or is part of. See property chain: a participant of a process at a given time is located where the process is located." ()
  • label "is included in" ()

Property Characteristics (1)

  • Transitive ('is included in')

Inverses (1)

  • includes

Usage (10)

  • ObjectProperty: 'is included in'
  • canonicity ⊆ 'is included in' only (function or 'immaterial object' or 'material object' or process)
  • 'entity at some time' ⊆ 'is included in' only 'entity at some time'
  • 'information object' ⊆ 'is included in' only ('information object' or 'material object')
  • 'material object' ⊆ 'is included in' only 'entity at some time'
  • quality ⊆ 'is included in' only ('immaterial object' or 'material object' or process or quality)
  • 'immaterial object' or 'information object' or 'material object' and not 'entity at some time' ⊆ not ('is included in' some thing)
  • 'inheres in' ⊆ 'is included in'
  • 'is part of' ⊆ 'is included in'
